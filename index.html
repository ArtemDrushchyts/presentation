What is React?

-React is an open source JavaScript library for developing user interfaces

-React can be used to develop single-page and mobile applications. Its goal is to provide high speed, simplicity and scalability

How does React Work?

  • React creates a VIRTUAL DOM in memory
  • React only changes what needs to be changed

LIFECYCLE METHODS

constructor (props): constructor in which the initialization of the component takes place.

componentWillMount (): called just before rendering the component.

render (): component rendering.

componentDidMount (): called after component rendering. Here you can make requests to remote resources.

componentWillUnmount (): called before removing a component from the DOM.

React Hooks

Hooks are a new addition in React 16.8. They let you use state and other React features without writing a class.
